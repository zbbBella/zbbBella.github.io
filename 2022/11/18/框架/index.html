一、Spring简介

初识Spring

  • 官网：spring.io

  • 全家桶

    核心要学习的：

    Spring Framework

    Spring Boot

    Spring Cloud

Spring 发展史

现在是5.0版本

Spring Framework系统架构

  • Data Access 数据访问
  • Data Integration 数据集成
  • Web Web开发
  • AOP 面向切面编程
  • Aspects AOP思想实现
  • Core Container 核心容器
  • Test 单元测试与集成测试

核心概念

IoC/DI（Inversion of Control）

  • 使用对象时，主动new => 外部提供对象
  • 对象创建控制权由程序转移到外部 => 控制反转思想

IoC容器

  • Spring提供了一个容器：IoC容器，充当IoC思想中的“外部”
  • IoC容器负责对象的创建、初始化等一系列工作

Bean

  • 被创建or管理的对象在IoC中统称为Bean

目标：充分解耦

  • 使用IoC容器管理bean
  • 在IoC容器内将有依赖关系的bean进行关系绑定（DI）

最终效果

  • 适用对象的时候不尽快与直接从IoC容器中获取，并且获取到的bean已经绑定了所有的依赖关系

bean配置

  • bean默认为单例：节省空间
  • 适合交给容器管理的bean
    • 表现层对象 server
    • 业务层对象 service
    • 数据层对象 dao
    • 工具类对象
  • 不适合交给容器管理的bean
    • 封装实体的域对象

bean生命周期

  • 初始化容器
    1. 创建对象（内存分配）
    2. 执行构造方法
    3. 执行属性注入（set）
    4. 执行bean初始化方法
  • 使用bean
    • 执行业务操作
  • 关闭or销毁容器
    • 执行bean销毁方法
